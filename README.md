# Estudos de Linguagem C

Repositório de estudos e experimentos com **matrizes 2D em C**, baseado no curso **"Linguagem C – O Curso Definitivo"** do Wagner Rambo (WR Cursos).  

[Curso no Hotmart](https://hotmart.com/pt-br/marketplace/produtos/linguagem-c-o-curso-definitivo/N48026554Y)


## Objetivo

- Praticar conceitos básicos de programação em C
- Entender como percorrer, modificar e exibir matrizes
- Aprender a organizar código e validar entradas do usuário
- Servir como registro de estudos do curso
